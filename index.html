About Reave

Reave is an innovative media solution for sharing stories and writings built on the Arweave Blockchain. Get benefits and benefits by using Reave.

Technology

Reave uses the existing Arweave Blockchain. This blockchain allows storing all story and writing data into blocks and nodes permanently.

Permanently

Unlike other platforms, Reave will keep the stories you have written forever. You don't need to be afraid if your story disappears, because that is not possible.

Incentive

Story writers, developers, and investors have the opportunity to get incentives from activities carried out by all users using Arweave tokens.

User Friendly

Reave is designed with a user-friendly interface. Reave while maintaining comfort when writing and sharing stories. This very fun fo all.

Tokenization

Reave implements an intelligent contract called SmartWeave to create a new token, the Reave token with the REV Ticker (Reave Token).

Open Source

The source code is available on Github. Developers can find it as a reference and provide feedback for Reave as a feature enhancement in the future.

PROBLEM

In the first quarter of 2020, internet users around the world reached 4.5 trillion active users. Among these statistics, some of them are users of online publishing media. One problem that is seen today is that the publish platform is still centralized.

All writers and readers are subject to and obey the rules and regulations both from the agency and from the government. This allows the deletion of data and author's account so that it can be detrimental to the author. In addition, there is still a lack of platforms that provide income directly from readers to writers.

SOLUTION

Reave's story and information sharing platform is intended so that the Author has the freedom to write and publish his writings on the internet. Because it is decentralized, it means the user account is 100% owned by the user.

All asset, data and information management is managed by the user himself. In other words, Author is free to delete, post and share any information on the Reave platform. Reave is permanent, meaning that all user account data will not be lost in the blockchain ledger. All are recorded and stored on the Arweave Blockchain. In addition, the best solution is to provide 100% incentives to Author.
